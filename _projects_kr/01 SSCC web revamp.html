Technologies Used

  • python
  • BeautifulSoup
  • selenium
  • Web crawl

My Role

UW-Madison 의 Social Science Computing Cooperative 에 저는 다량의 HTML 파일을 parsing 하여 XML file 로 추출하는 python 코드를 제공 하였습니다. 이로 인해 약 200개 가량의 문서 이동을 자동화하여 Web Revamp 프로젝트를 빠르게 완료할 수 있었습니다.

Project Description

프로그램은 HTML 문서들을 KB 사이트에 이주를 할 수 있도록 XML 파일로 전환을 시켜줍니다. 그 이전에 프로그램은 https://ssc.wisc.edu 를 web crawl 하여 그 도메인 안에 문서들을 전부 저장하고 필요한 형식에 맞도록 parsing 을 합니다.

Project Difficulty

web migration 이라는 작업은 이전 경험 잆이 진행하게 되면 굉장히 까다로운 작업입니다. 예를 들어, 문서간의 링크가 깨지지 않기 위해서 도메인의 문서들간 맵핑이 잘 되어있어야 하는 것 등 여러 주의사항들이 있습니다.

My Solution

저는 web migration 에 대한 자료들을 참고하여, 프로젝트를 늦추는 불상사가 생기지 않도록 여러 주의사항들 인지하였습니다. 또한, 이러한 정보들을 회사 동료들에게 전달하여 보다 나은 해결책을 제공할 수 있었습니다.
