My Role

저는 프로젝트의 계획과 프로그램 플로우 디자인을 하였습니다. 또한, 그래프 시각화, threading, UI 를 포함하는 GUI 부분을 담당하였습니다.

Project Description

Algotrade 는 암호화폐 거래를 유저가 선택한 알고리즘으로 자동화 시켜주는 python 프로그램 입니다. 프로젝트는 더욱 다양한 알고리즘을 위하여 추후에 계속 진행될 예정입니다.

Project Difficulty and Solution

점점 기능을 추가시키며 개발할수록 스케일이 커지는 프로젝트를 통제하는 것은 쉽지 않았습니다. 특히나, file control 과 code tracing 은 시간소요가 컸으며, 이전 구성에 조정이 필요하였습니다. 이를 해결하기 위해 저희는 프로젝트를 더욱 모듈화 시켜서 신속한 조정이 어느때나 가능하도록 하였습니다.
