Project Description

인터넷에는 정말 많은 데이터들이 존재합니다. 하지만, 데이터가 Neural Network 에 사용 되기 위해서는 data cleaning 필요합니다. clean data 의 부족을 해결 하기 위해, 저는 hierarchical clustering 알고리즘을 PCA 와 사용하여 컴퓨터가 데이터를 labeling 할 수 있도록 하였습니다.

Limitation to Note

이 프로젝트의 결과물은 저의 기대를 미치지 못했습니다. 오직 소수의 기존 데이터 만이 프로그램에 의해 labeling 이 될 수 있었습니다. 또한, PCA 가 data transformation 단계에서 생성하는 모호성 때문에 몇 컴퓨터의 classification 은 사람에게 의미가 없을 수 있습니다.

Result

비록 프로젝트가 저의 기대에 미치지지는 않았지만, 저는 머신러닝 문제를 과학적으로 접근하는 방법에 대해 배웠습니다. 그리고 이 프로젝트는 머신러닝을 향한 저의 여정의 시작에 불과합니다.
