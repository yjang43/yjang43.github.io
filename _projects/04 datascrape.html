Project Description

There are so much data in the Internet, and yet they need data cleaning to be used in a Neural Network. To overcome the lack of clean data, I used hierarchical clustering algorithm with PCA to let machine label data.

Limitation to Note

The result of the project did not reach my expectation. Only the small portion of the original data was able to be labeled by the program. Also, due to the ambiguity that data transformation (PCA) creates, the arbitrary classification becomes insensible to human.

Result

Although the project did not match to my expectation, I learned a scientific approach to ML problems, and this project is only the beginning of my journey to ML.
