My Role

To Social Science Computing Cooperative, UW-Madison, I provided a python code to parse and export numerous HTML files into an XML file. This allowed to automate and fasten the process of migrating around 200 documents.

Project Description

The script reformats HTML documents into XML so that they can be migrated to a new KB site. Prior to this, the script crawled https://ssc.wisc.edu to receive all the documents within the domain, and then parsed to fit the required format.

Project Difficulty

Web migration is a tricky operation to first-comer of such project because the project always sits on the risk of starting from the beginning if one does not know what's ahead. For example, we needed to preserve the mapping of our website because links within each document breaks without a proper mapping.

My Solution

There were many articles regarding the web migration, so I read to prevent us from slowing down the project. Also, I addressed the possible issues to my coworkers and mostly was able to find a better solution. through out communication.
